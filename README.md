# Deploying a Machine Learning Model with FastAPI

Classification model on publicly available Census Bureau data, with model
performance monitoring on data slices, deployed via FastAPI.

This repository is an OUTLINE / scaffold. Each file contains stubs, docstrings,
and `# TODO` comments mapped to the project steps and rubric.

## Project Steps & Files

- Step 1 - GitHub Action: `.github/workflows/ci.yml`
  (runs pytest + flake8 on push; screenshot -> `screenshots/continuous_integration.png`)
- Step 2 - Data: `ml/data.py` (`process_data`); place `census.csv` in `data/`
- Step 3 - Model: `ml/model.py`
  (`train_model`, `inference`, `compute_model_metrics`, `save_model`,
  `load_model`, `performance_on_categorical_slice`)
- Step 4 - ML Pipeline: `train_model.py` (writes `slice_output.txt`)
- Step 4 - Unit Test: `test_ml.py` (>= 3 tests; screenshot -> `screenshots/unit_test.png`)
- Step 4 - API: `main.py` (FastAPI GET + POST)
- Step 4 - API interaction: `local_api.py` (screenshot -> `screenshots/local_api.png`)
- Step 4 - Model card: `model_card.md`
- Step 5 - Push to GitHub: commit and push, confirm the Action passes.

## How to run

```
pip install -r requirements.txt
python train_model.py          # trains, saves model/encoder, writes slice_output.txt
pytest test_ml.py -v           # run unit tests
uvicorn main:app --reload      # start the API
python local_api.py            # exercise the API (GET + POST)
flake8 .                       # lint
```
